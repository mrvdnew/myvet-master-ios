## Color Reference

### Primary Colors
- **Primary Green**: `Color(red: 0.30, green: 0.63, blue: 0.31)`
- **Secondary Orange**: `Color(red: 1.0, green: 0.60, blue: 0.0)`
- **Background**: `Color(red: 0.96, green: 0.94, blue: 0.92)`

### Status Colors
- **Scheduled**: Green `Color(red: 0.30, green: 0.63, blue: 0.31)`
- **Confirmed**: Cyan
- **Completed**: Gray
- **Cancelled**: Red

### Text Colors
- **Primary Text**: Black
- **Secondary Text**: Gray

### Component Styling
- **Button Height**: 50pt
- **Corner Radius**: 8-12pt
- **Card Padding**: 16pt
- **Section Spacing**: 24pt

---

## Responsive Behavior

### Cards
- White background
- 12pt corner radius
- Shadow: `rgba(0,0,0,0.05)` with 4pt radius
- 16pt internal padding

### Input Fields
- White background
- 8pt corner radius
- 1pt gray border
- 16pt horizontal padding
- 14pt vertical padding

### Buttons
- Primary: Green background, white text
- Secondary: Gray background, black text
- Danger: Red background, white text
- Height: 50pt
- Corner radius: 8pt
- Full width when in forms

### Navigation
- Tab bar: White background
- Active tab: Green icon
- Inactive tab: Gray icon
- Badge support for notifications
